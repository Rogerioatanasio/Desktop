# Projeto Azul 🚧🔧

Plataforma simples para conectar profissionais de obra
(pedreiros, eletricistas, encanadores, pintores, faxineiras)
com pessoas que precisam contratar serviços.

Este projeto é um **MVP em fase de testes**, criado para validar
a ideia na prática antes de escalar funcionalidades.

## Funcionalidades atuais
- Cadastro de profissionais
- Listagem de profissionais disponíveis
- Filtro por profissão
- Contato direto via WhatsApp

## Status do projeto
🧪 MVP em teste real com usuários
